CAD - Computer-Aided Design

Mon, Dec 13, 2021 6:00 PM - 7:00 PM EDT

AutoCAD

In our AutoCAD Essentials course, you will learn the core topics required for preparing professional drawings using AutoCAD and learn to navigate the AutoCAD workspace and use basic drawing, editing, and viewing tools of the software. You will organize drawing objects on layers and insert reusable blocks. Prepare layouts to plot and add text, hatching, and dimensions. By the end of the course, you will complete a hands-on project that can be used as a foundation for your professional portfolio.

In our AutoCAD Intermediate course, you will be able to better utilize productivity tools and learn how to work with constraints, create and organize more advanced blocks and layouts as well as build templates and annotation styles from scratch, and learn how to use zoom commands, external references, QuickCalc, and dimensioning tools for a more complete and advanced drawing.

In our AutoCAD Advanced course, you will advance your skills in the AutoCAD workspace and gain the final pieces of knowledge you need to pursue the Autodesk certification after the class. Learn to work with advanced tables, annotations, editing, and viewing tools of the AutoCAD software. You will create advanced dynamic blocks and attributes and learn about sheet sets. Prepare to publish and use collaboration tools all while managing layers and learning to use the right tools for optimal performance. You will complete a hands-on project in this program that can be used in your professional portfolio.

Revit Architecture

In our Revit Architecture courses, you will learn to create a full 3D architectural project model from the ground up using Autodesk Revit, utilize basic tools that the majority of architectural users need to create walls, doors, windows, floors, ceilings, roofs, and stairs, and navigate the Revit user interface and use the basic drawing, editing, sketching, modifying, and viewing tools of the software. You can discover the purpose of BIM and how it is applied in the Revit software. By the end of the course, you will be able to create levels and grids as datum elements for a model, add component features, such as furniture and equipment, and set up sheets for plotting with text, dimensions, details, tags, and schedules.
