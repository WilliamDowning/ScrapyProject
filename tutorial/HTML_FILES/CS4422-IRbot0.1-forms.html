    Prerequisite Override Request for SCI 1102

    If you need a prerequisite override because you are currently enrolled in the prerequisite class somewhere else (transient, transfer, cross-registration), you will need to email the appropriate advisor with your name, KSU ID number, the course(s) you would like the override for, and proof of the prerequisite either by way of unofficial transcript screenshot or CollegeBoard AP scores screenshot. You cannot use this form.

    Please read these notes and policies before submitting a request. Your request will be ignored for inability to read and follow directions.

    1. This form should be used only for SCI 1102.
    2. THIS IS NOT A CLOSED CLASS OVERRIDE form. We do not allow closed class overrides.
    3. The requests will be checked twice a week. You WILL NOT receive an email when the override is processed. Instead, you can check your "Registration Status and Time Ticket" screen under the "Registration" tab of Owl Express. If your request is not approved, you will receive an email.
