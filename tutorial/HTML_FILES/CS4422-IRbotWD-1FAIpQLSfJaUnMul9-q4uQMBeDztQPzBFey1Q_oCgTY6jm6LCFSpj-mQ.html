Missing Titles Form
FIRST: Ask for assistance at the library check out desks if you are unable to find a title in its location.

If a title is not found, then please use this form (1 title per form). If you need the title, then use GIL-Express or Interlibrary Loan services to request it from another library.  http://libanswers.kennesaw.edu/ill/faq/198116

 Access Services staff will conduct a search for the title quarterly for up to one year. If the title is not found after one year, then it will be declared lost.
