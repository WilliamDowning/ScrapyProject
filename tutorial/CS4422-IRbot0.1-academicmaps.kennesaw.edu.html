Academic Program Maps

Academic program maps offer suggested four-year schedules of courses for all of KSU's degree programs based on requirements in the Fall 2020 KSU catalog. Each map provides a sample schedule each term, including milestones, keeping you on track for timely progress to graduation! Questions regarding the academic program maps may be sent to askadvising@kennesaw.edu. If you would like to request a change to a program map that is not associated with a curriculum change, please use this form.
